# GLAE

Esta biblioteca é voltada para o aprendizado de programação básica em sala de aula. Ela tem seu foco em comandos como os de seleção de condicionais e laços, e também nas estruturas básicas como vetores, matrizes, strings (palavras ou textos) e vetores dinâmicos. A biblioteca usa os conceitos de visualização para facilitar a compreensão do processo que o sistema percorre para gerar certo resultado. Estes conceitos são utilizados através do uso de elementos gráficos desenvolvidos usando a biblioteca OpenGL. O acesso a esses elementos gráficos é facilitado pelo uso dessa biblioteca. Também foram criados materiais para uso pelos alunos como: documentação e guia de uso, atividades práticas e exemplos de uso.
